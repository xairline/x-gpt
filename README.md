# X-GPT

## Overview
This project enables the integration of OpenAI's ChatGPT 
with the popular flight simulator X-Plane. By 
leveraging the advanced language model capabilities 
of ChatGPT, users can interact and control 
various aspects of the X-Plane flight simulator 
using natural language commands.

## Features
- **Natural Language Control** 

    Issue commands to X-Plane in natural language, such as "increase altitude" or "set course for JFK airport".
 
- **Flight Planning Assistance**
  
    ChatGPT can assist in creating flight plans based on user input.

- **Real-Time Flight Monitoring**

    Monitor flight parameters and receive updates through ChatGPT.
